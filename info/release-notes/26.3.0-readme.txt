New / Improvements
==================

* Equipment
  https://github.com/mytourbook/mytourbook/issues/777
  
  Equipment and tags have almost equal features but tags have been "misused" for an equipment because of missing a real equipment.
  These are the feature which do not have the other:

  - Equipment
    - Equipment items are created directly in the equipment view
    - Parts and services are collating tours by a type and time until the next or previous part or service

  - Tags
    - Categorize tags
    - Floating tag context menu
    

* Tour Blog
  - Display time for each tour blog entry
    https://github.com/mytourbook/mytourbook/issues/1602
