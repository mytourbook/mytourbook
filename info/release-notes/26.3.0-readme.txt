New / Improvements
==================

* Equipment
  https://github.com/mytourbook/mytourbook/issues/1590
  
  Equipment and tags have a lot of common features but tags have been "missused" for an equipment because of missing equipment.
  These are the feature which do not have the other:
  
  - Tags
    - Categorize tags
    - Floating tag context menu
    
  - Equipment
    - Equipment items are created directly in the equipment view
    - Parts and services are collating tours by a type and time

* Tour Blog
  - Display time for each tour blog entry
    https://github.com/mytourbook/mytourbook/issues/1602
