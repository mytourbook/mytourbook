Changes
=======

* Equipment
  - Changed wording from "Type" to "Collate ID"
  - Reordered fields in the equipment/part and service dialogs

* HR-Zones
  - When HR-Zones are modified in the preferences, these modifications cannot be applied directly in the preferences. There are new actions in the Tour Book view context menu "Adjust Tour Values/HR Zones" to apply these modifications to all or the selected tours.
    https://github.com/mytourbook/mytourbook/issues/1710
  
* Eclipse 2026-06
  - When using the command line parameter, e.g. -Dswt.autoScale=200, this will not work anymore with an integer value. Additionally this command line parameter must also be set: -Dswt.autoScale.updateOnRuntime=false
    https://eclipse.dev/eclipse/markdown/?f=news/4.39/platform_isv.md#monitor-specific-scaling-by-default-windows
