New / Improvements
==================

* Tour filter 
  - Added nutrition tour filter (Frederic)
  - Added option in the equipment tour filter to exclude tours with an equipment
    https://github.com/mytourbook/mytourbook/issues/1641

* Tour Nutrition (Frederic)
  - Select a product from a list of the most recent products used

* Equipment
  - Add a contextual menu to add several equiments at a time
    https://github.com/mytourbook/mytourbook/issues/1657
    
* Tagged Tours View
  - New columns to display the parent of a tag and how tours are sorted/categorized
  

Changes
=======

* Easy Import
  - Changed launcher layout
  - Cadence settings needs to be recreated, they are not related any more with the tour type configurations

* Tour Nutrition (Frederic)
  - The update button is now in the contextual menu of each product and is limited to 1 product at a time since the OpenFoodFacts database is very limited

* Equipment
  - Multiple equipment with an empty type are not collated any more, each equipment with an empty type can now collate tours for itself
    Update your equipment by removing the type when this equipment/part is not depending on another with the same type
    The new column "Type (saved)" can display the saved value which is a random value when the type in the UI is empty.
  - The equipment tour filter operator is now supporting also the NOT operator. 
    This needed to rewrite the code and the old OR or AND operator have to be checked.
  

Bugfix
======
  
* Tour Nutrition (Frederic)
  - Fixed an exception when updating the list of products for which a product is not found in the OpenFoodFacts database
  - Fixed the computation of total calories/carbs/sodium for beverages involving beverage containers
  - Fixed a bug where modifying a custom product would remove the associated beverage container
